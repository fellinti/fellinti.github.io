Technology Research Paper

Duo Security

For this assignment I interviewed my friend Robbie Small who works as a software engineer at the company Duo Security based in Ann Arbor, Michigan. Duo Security is a company that provides two factor authentication for over 5000 organizations including Facebook, Twitter, Yelp, and Etsy.

Web and Database Servers

Multiple Amazon Web Services (AWS) instances for database and Web Application deployment
Redis for Key/Value Storage
Web Server - Google app engine for static website

Special Software

Salesforce for metrics
Phabricator/Confluence for internal use (picking up tasks to work on/internal documentation)
Twilio API for sending out SMS messages and phone calls
Google Cloud Messaging for Push service on Android and Apple Push Notifications for iOS

Open Source Software/Hardware

RX Java (https://github.com/ReactiveX/RxJava)for functional programming in the Android App
Yubikey for 2 Factor Hardware
Duo Security also has their own open souce APIs for most languages/platforms (https://github.com/duosecurity)

Deployment Strategies

Release in 2 week “sprints” for the main product
Multiple AWS instances in case one goes down or is overloaded
Google app engine for static content (duosecurity.com)

Security

Heavily reliant on two factor services
Cannot authenticate without a second factor
Push Notification, HOTP (HMAC-based one time passcodes), TOTP (Time-based one time passcodes) or phone callback to verify identity
Must sign into the Juniper VPN for development
Lots of audit scripts code must pass before building the product
